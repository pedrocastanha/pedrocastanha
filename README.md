### What I work on

- **Agent runtimes** — subagents, skills, MCP, plan mode, tool permissions
- **Observability** — lightweight tracing for LLM/agent graphs (tokens, cost, latency, tools)
- **Production AI** — cost control, multi-tenant backends (NestJS + FastAPI + Postgres)
- **Devtools** — open-source CLIs and SDKs people can install and run
